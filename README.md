# Resumen de etiquetas HTML

Material educativo para la materia **"Aplicaciones de Diseño, turno noche 2026"**
del **Instituto Técnico Boliviano Suizo (TBS)** — Potosí.

## ¿Qué contiene?

Un documento HTML interactivo que resume las etiquetas HTML más utilizadas,
organizadas en 10 categorías:

1. Estructura básica del documento
2. Etiquetas de texto
3. Enlaces e imágenes
4. Listas
5. Tablas
6. Formularios
7. Etiquetas semánticas HTML5
8. Multimedia
9. Contenedores y organización
10. Etiquetas útiles modernas

Cada sección incluye una tabla de referencia y un ejemplo práctico dividido en
dos columnas: resultado visual a la izquierda y código fuente a la derecha.
